## 🌌 🚀 Overview

**Social Persona Simulator** is a futuristic interactive system that allows users to **simulate different social personalities** and observe how behavior, tone, and outcomes change in real-time.

It acts like a **psychological engine + social AI**, helping you understand:

* 🧠 How different personalities respond
* 🎭 How tone affects outcomes
* 📊 How context changes behavior

> 💡 *You don’t have one personality… you have many.*
